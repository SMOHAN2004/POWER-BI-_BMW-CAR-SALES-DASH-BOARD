# BMW Sales Dashboard (2010-2024)

## Project Objective
Analyze BMW's worldwide sales trends from 2010 to 2024 to identify key regions, top-performing models, and evolving fuel preferences. Provide data-driven insights to support strategic decisions in market expansion, electrification strategy, and model optimization.
